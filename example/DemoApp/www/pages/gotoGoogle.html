Go to Google!

Google Maps for mobile has a view camera. You see the map through the camera. This plugin provides animateCamera() and moveCamera() methods.

Map.animateCamera() method changes the camera view with animation.

JavaScript

const GOOGLE = new plugin.google.maps.LatLng(37.422858, -122.085065); const GOOGLE_TOKYO = new plugin.google.maps.LatLng(35.660556,139.729167); map.moveCamera({ 'target': GOOGLE, 'zoom': 0 }, function() { setTimeout(function() { map.animateCamera({ 'target': GOOGLE, 'tilt': 60, 'zoom': 18, 'bearing': 140 }); }, 2000); });
