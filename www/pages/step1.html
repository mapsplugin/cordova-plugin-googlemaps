Step1: Display a map

Let's display a map in the app. The code is very simple like this:

document.addElement("deviceready", function() {
  // Intiailize the map instance.
  var map = plugin.google.maps.Map.getMap(mapDiv);
  
  // Wait until the map is ready.
  map.on(plugin.google.maps.event.MAP_READY, function() {
    // Google Map is displayed.
  });
});
